Imgdump is a lightweight web application for uploading and sharing images and videos. Users can register, log in, and upload media directly to the site. The gallery displays uploads with timestamps and the uploader's username.  

---

## Features

- User registration and login system.
- Upload images (`jpg`, `jpeg`, `png`, `gif`, `webp`) and videos (`mp4`, `webm`, `ogg`).
- Gallery with tabs to switch between images and videos.
- Shows who uploaded each file and the upload timestamp.
- Pagination for large galleries.
- Responsive gallery items with fixed size using `object-fit`.
- Optional AJAX uploads for smoother user experience.
- Simple, colorful UI built with HTML, CSS, and PHP.
